# how to
```
$ git clone https://github.com/kazhr/config.git ${local_repo_dir}
```

## setup ansible
```
$ cd ${loca_repo_dir}/ansible
$ ./install
```

## setup awscli
```
$ cd ${loca_repo_dir}/aws
$ ./install
```

## run django
```
$ cd ${loca_repo_dir}/django
# docker-compose up
```

## setup docker
```
$ cd ${loca_repo_dir}/docker
$ ./install
```

## setup dotfiles
```
$ cd ${loca_repo_dir}/dotfiles
$ ./install
```

## setup nvim
```
$ cd ${loca_repo_dir}/nvim
$ ./install
```
